## Effortlessly Self-host Agnost Using its Helm Chart

```console
helm repo add agnost-gitops https://cloud-agnost.github.io/agnost-gitops-charts/
helm upgrade --install agnost agnost-gitops/base --namespace agnost --create-namespace
```

View Agnost [documentation](/getting-started) for detailed set-up and configuration instructions and platform guides.
